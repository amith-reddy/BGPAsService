Introduction

  • ExaBGP Portal is the Frontend for ExaBGP Tool.
  • ExaBGP Tool is a open source tool developed to allow engineers to control their network from a single server. This can be termed as Software Defined Networking of BGP.
  • ExaBGP Portal allows engineer to configure BGP Flows and push the flows to the BGP neighbors. It also supports scheduling of the flows at a scheduled time.
  • A Notification system is implemented to notify engineers through email in case of any failures while pushing flow rules to the BGP neighbors.
